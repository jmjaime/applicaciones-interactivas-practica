Estructura Básica de un Documento HTML

Ejemplo 1: Anatomía de un documento HTML5 completo

Elementos Fundamentales

1. DOCTYPE

<!DOCTYPE html> - Declara que este es un documento HTML5

2. Elemento html

<html lang="es"> - Contenedor raíz con idioma especificado

3. Head - Metadatos

  • meta charset="UTF-8" - Codificación de caracteres
  • meta name="viewport" - Configuración para dispositivos móviles
  • title - Título que aparece en la pestaña

4. Body - Contenido Visible

Todo el contenido que ves en esta página está dentro del body

Lo que Ignoran los Navegadores

Los navegadores ignoran:

  • Espacios múltiples (se convierten en uno solo)
  • Saltos de línea en el código fuente
  • Tabulaciones
  • Comentarios HTML

Este texto tiene muchos espacios entre palabras, pero el navegador solo muestra uno.

Elementos Vacíos

Algunos elementos no tienen contenido ni etiqueta de cierre:

  • <br /> - Salto de línea
  • <hr /> - Línea horizontal
  • <img /> - Imagen
  • <meta /> - Metadatos
