Familias de Fuentes

Serif

Las fuentes serif tienen pequeños trazos decorativos en los extremos de las letras. Son tradicionales y elegantes, ideales para textos largos impresos. Ejemplos: Times New Roman, Georgia, Playfair Display.

Sans-serif

Las fuentes sans-serif no tienen trazos decorativos. Son modernas y limpias, perfectas para pantallas digitales. Ejemplos: Arial, Helvetica, Inter.

Monospace

Las fuentes monospace tienen el mismo ancho para cada carácter. Perfectas para código y datos tabulares. Ejemplos: Courier New, Fira Code, Monaco.
/* Definir familias de fuentes */
.serif { font-family: "Playfair Display", Georgia, serif; }
.sans-serif { font-family: "Inter", Arial, sans-serif; }
.monospace { font-family: "Fira Code", "Courier New", monospace; }

Diferencia clave:
  • em: Relativo al font-size del elemento padre
  • rem: Relativo al font-size del elemento raíz
  • vw/vh: Relativo al viewport (1vw = 1% del ancho)

Tipografía Responsive

Este texto se adapta al tamaño de la pantalla usando clamp()
.responsive-text {
  font-size: clamp(1rem, 4vw, 2.5rem);
  /* min: 1rem, preferido: 4vw, max: 2.5rem */
}

Mejores Prácticas

Legibilidad

  • Contraste: Mínimo 4.5:1 para texto normal
  • Line-height: Entre 1.4 y 1.6 para párrafos
  • Longitud de línea: 45-75 caracteres óptimo
  • Tamaño mínimo: 16px para texto base

Performance

  • Usar font-display: swap para fuentes web
  • Precargar fuentes críticas con rel="preload"
  • Limitar el número de pesos y estilos
  • Considerar fuentes del sistema para mejor rendimiento

Ejercicio: Cambia las fuentes en DevTools y observa cómo afecta la legibilidad y el diseño.
