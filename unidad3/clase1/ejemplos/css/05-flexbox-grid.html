/* Contenedor Flex */
.flex-container {
  display: flex;
  flex-direction: row; /* row, column, row-reverse, column-reverse */
  justify-content: center; /* flex-start, center, flex-end, space-between, space-around, space-evenly */
  align-items: center; /* flex-start, center, flex-end, stretch, baseline */
  gap: 1rem;
}

/* Elementos Flex */
.flex-item {
  flex-grow: 1; /* proporción de crecimiento */
  flex-shrink: 1; /* proporción de encogimiento */
  flex-basis: auto; /* tamaño base */
}

/* Contenedor Grid */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */
  grid-template-rows: 100px auto 50px;
  gap: 1rem;
  grid-template-areas:
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

/* Elementos Grid */
.grid-item {
  grid-column: 1 / 3; /* de columna 1 a 3 */
  grid-row: 2 / 4; /* de fila 2 a 4 */
  grid-area: header; /* usar área nombrada */
}

Flexbox vs Grid: ¿Cuándo usar cada uno?

Aspecto Flexbox CSS Grid
Dimensión Unidimensional (fila o columna) Bidimensional (filas y columnas)
Uso Principal Componentes, alineación Layouts de página
Flexibilidad Elementos flexibles Posicionamiento preciso
Mejor para Navegación, botones, centrado Layouts complejos, grillas

Recomendación

Usa Flexbox para: Componentes individuales, alineación de elementos, navegación.
Usa Grid para: Layouts de página completa, sistemas de grillas, posicionamiento complejo.

  display: flex;
  justify-content: space-between;
  align-items: center;
}

/* Cards con Grid */
.card-layout {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1.5rem;
}

/* Card individual con Flexbox */
.card {
  display: flex;
  flex-direction: column;
}

.card .button {
  margin-top: auto; /* empuja el botón al final */
}
