Diferencia importante:
  • opacity: Afecta todo el elemento (contenido incluido)
  • rgba/hsla: Solo afecta el color específico
/* Opacidad vs transparencia */
.elemento-opaco { opacity: 0.5; } /* Todo el elemento */
.fondo-transparente { background-color: rgba(255, 0, 0, 0.5); } /* Solo el fondo */

Consejos para Paletas de Colores

  • Regla 60-30-10: Color dominante (60%), secundario (30%), acento (10%)
  • Contraste: Asegurar legibilidad con ratios adecuados
  • Consistencia: Usar variables CSS para mantener coherencia
  • Accesibilidad: Probar con simuladores de daltonismo

Variables CSS para Colores

:root {
  /* Definir paleta de colores */
  --color-primary: #3498db;
  --color-secondary: #2ecc71;
  --color-accent: #e74c3c;
  --color-neutral: #95a5a6;
  --color-dark: #2c3e50;
  --color-light: #ecf0f1;
}

.button {
  background-color: var(--color-primary);
  color: var(--color-light);
}

.button:hover {
  background-color: var(--color-accent);
}

Ventajas de las Variables CSS

  • Mantenimiento: Cambios centralizados
  • Consistencia: Colores uniformes en todo el sitio
  • Temas: Fácil implementación de modo oscuro/claro
  • Reutilización: Valores compartidos entre componentes
