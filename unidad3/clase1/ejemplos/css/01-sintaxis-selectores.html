Especificidad CSS

Demostración de Especificidad

Este texto debería ser rojo porque el selector de ID tiene mayor especificidad que las clases.

.especificidad-demo { color: blue; } /* 0,0,1,0 */
#especificidad-id { color: red; } /* 0,1,0,0 - GANA */
.especificidad-demo.clase-extra { color: green; } /* 0,0,2,0 */

Orden de Especificidad

  1. Inline styles (1,0,0,0)
  2. IDs (0,1,0,0)
  3. Classes, attributes, pseudo-classes (0,0,1,0)
  4. Elements y pseudo-elements (0,0,0,1)

Agrupación de Selectores

Títulos con Fuente Serif

Los selectores h1, h2, h3, .titulo-especial comparten la misma familia de fuente (Georgia, serif).

h1, h2, h3, .titulo-especial {
  font-family: Georgia, serif;
}
