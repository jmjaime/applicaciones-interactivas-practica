Indicadores de Carga

/* Keyframes */
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(20px); }
  to { opacity: 1; transform: translateY(0); }
}

/* Aplicar animación */
.fade-in {
  animation: fadeIn 2s ease-out;
}

/* Transiciones */
.hover-effect {
  transition: all 0.3s ease;
}

.hover-effect:hover {
  transform: scale(1.1);
}

Elementos Interactivos

Botones con Efectos

Mejores Prácticas

  • Performance: Usar transform y opacity para animaciones suaves
  • Accesibilidad: Respetar prefers-reduced-motion
  • UX: Animaciones sutiles y con propósito
  • Duración: 200-500ms para micro-interacciones
/* Variables CSS para consistencia */
:root {
  --transition: all 0.3s ease;
  --shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

/* Respeto por preferencias de accesibilidad */
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

/* Soporte para modo oscuro */
@media (prefers-color-scheme: dark) {
  .section { background-color: #34495e; }
}

CSS Moderno

Características Modernas

  • CSS Variables: Theming dinámico y mantenimiento
  • clamp(): Tipografía y espaciado fluido
  • Grid y Flexbox: Layouts sin media queries
  • aspect-ratio: Proporciones consistentes
  • Container Queries: Responsive basado en contenedor
/* CSS Variables */
:root {
  --primary: #3498db;
  --spacing: clamp(1rem, 4vw, 2rem);
}

/* Funciones modernas */
.element {
  width: min(90vw, 1200px);
  padding: var(--spacing);
  aspect-ratio: 16 / 9;
}

/* Container Queries (futuro) */
@container (min-width: 400px) {
  .card { display: flex; }
}
