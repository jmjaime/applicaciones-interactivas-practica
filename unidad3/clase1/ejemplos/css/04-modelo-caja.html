Componentes del Box Model

  • Content: El contenido real del elemento (texto, imágenes)
  • Padding: Espacio interno entre el contenido y el borde
  • Border: El borde que rodea el padding y el contenido
  • Margin: Espacio externo que separa el elemento de otros
/* Anatomía del Box Model */
.elemento {
  width: 200px; /* Ancho del contenido */
  height: 100px; /* Alto del contenido */
  padding: 20px; /* Espacio interno */
  border: 5px solid black; /* Borde */
  margin: 15px; /* Espacio externo */
}
/* Ancho total = 200 + 40 + 10 + 30 = 280px */

Box-sizing: La Revolución del Box Model

content-box (tradicional)

200px × 100px
+ padding + border

Ancho total: 260px (200 + 40 + 20)

border-box (moderno)

200px × 100px
incluye todo

Ancho total: 200px (incluye padding y border)

/* Modelo tradicional */
.content-box {
  box-sizing: content-box; /* default */
  width: 200px; /* solo contenido */
}

/* Modelo moderno (recomendado) */
.border-box {
  box-sizing: border-box;
  width: 200px; /* incluye padding y border */
}

/* Aplicar a todos los elementos */
* { box-sizing: border-box; }
Recomendación: Usar box-sizing: border-box hace que el diseño sea más predecible y fácil de manejar.

Conceptos Importantes del Margin

  • Centrado horizontal: margin: 0 auto
  • Colapso de márgenes: Los márgenes verticales se colapsan
  • Márgenes negativos: Permiten superponer elementos
  • Margin auto: Distribuye el espacio disponible automáticamente
/* Técnicas útiles con margin */
.centrado { margin: 0 auto; } /* centrado horizontal */
.espaciado { margin: 2rem 0; } /* solo vertical */
.superpuesto { margin-top: -20px; } /* negativo */
.sin-margin { margin: 0; } /* resetear */

/* Valores de display */
.block { display: block; } /* ocupa todo el ancho */
.inline { display: inline; } /* solo el ancho del contenido */
.inline-block { display: inline-block; } /* híbrido */
.hidden { display: none; } /* no se muestra */
.invisible { visibility: hidden; } /* oculto pero ocupa espacio */

Overflow: Controlando el Desbordamiento

overflow: visible
Este contenido es más alto que el contenedor y se desborda visiblemente.
overflow: hidden
Este contenido es más alto que el contenedor pero se corta.
overflow: scroll
Este contenido es más alto que el contenedor y muestra barras de desplazamiento.
overflow: auto
Este contenido es más alto que el contenedor y muestra scroll solo si es necesario.
/* Valores de overflow */
.visible { overflow: visible; } /* default, se desborda */
.hidden { overflow: hidden; } /* corta el contenido */
.scroll { overflow: scroll; } /* siempre muestra scroll */
.auto { overflow: auto; } /* scroll solo si es necesario */

Mejores Prácticas del Box Model

Recomendaciones

  • Usar border-box: * { box-sizing: border-box; }
  • Reset CSS: Eliminar márgenes y padding por defecto
  • Unidades consistentes: Usar rem/em para escalabilidad
  • Espaciado sistemático: Usar múltiplos de un valor base
  • DevTools: Inspeccionar el box model visualmente

Errores Comunes

  • No usar border-box: Cálculos de ancho complicados
  • Colapso de márgenes: No entender cómo funcionan
  • Overflow oculto: Contenido cortado inesperadamente
  • Display incorrecto: Elementos que no se comportan como esperado
/* Reset básico recomendado */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* Sistema de espaciado */
:root {
  --spacing-xs: 0.25rem;
  --spacing-sm: 0.5rem;
  --spacing-md: 1rem;
  --spacing-lg: 2rem;
  --spacing-xl: 4rem;
}
