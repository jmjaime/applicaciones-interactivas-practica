🖼️ Ejercicio 5: Galería con CSS Grid

Layouts Bidimensionales + Responsive + Animaciones

📋 Descripción del Ejercicio

Diseña una galería de imágenes moderna y responsive usando CSS Grid. Implementarás layouts complejos, efectos hover avanzados y animaciones suaves para crear una experiencia visual impactante.

🎯 Objetivos de Aprendizaje

  • Dominar CSS Grid para layouts bidimensionales complejos
  • Implementar responsive design sin media queries
  • Crear efectos hover y animaciones CSS avanzadas
  • Aplicar técnicas de overlay y transformaciones
  • Combinar Grid con Flexbox para componentes híbridos

📁 Archivo Base: galeria-grid.html

Crea un archivo llamado "galeria-grid.html":

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Galería de Fotos</title>
    <style>
      :root {
        --grid-gap: 1rem;
        --border-radius: 8px;
        --transition: all 0.3s ease;
        --overlay-bg: rgba(0, 0, 0, 0.7);
      }
      /* Aquí irán tus estilos CSS */
    </style>
  </head>
  <body>
    <!-- Contenido de la galería -->
  </body>
</html>

🏗️ Tarea 1: Estructura HTML de la Galería

Crea la estructura para una galería completa:

  • Header: Título de la galería y filtros/categorías
  • Grid container: Contenedor principal de las imágenes
  • Grid items: 12-15 elementos de imagen con:
    • Imagen (usa https://picsum.photos/ para placeholders)
    • Overlay con información
    • Título y descripción
    • Categoría/tag
  • Modal: Para vista ampliada (opcional)
💡 Pista: Usa https://picsum.photos/400/300?random=1 para imágenes placeholder. Cambia el número final para diferentes imágenes.

🎨 Tarea 2: Grid Layout Principal

Implementa el layout principal con CSS Grid:

  • Grid container: display: grid
  • Columnas responsive: grid-template-columns: repeat(auto-fit, minmax(250px, 1fr))
  • Filas automáticas: grid-auto-rows: 200px
  • Gap: Espaciado entre elementos
  • Elementos destacados: Algunos items ocupan más espacio

💡 Ejemplo de Grid Responsive:

.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  grid-auto-rows: 200px;
  gap: var(--grid-gap);
  padding: 2rem;
}

/* Elementos destacados */
.gallery-item.featured {
  grid-column: span 2;
  grid-row: span 2;
}

.gallery-item.wide {
  grid-column: span 2;
}

🖼️ Tarea 3: Elementos de la Galería

Estiliza cada elemento de la galería:

  • Contenedor: position: relative, overflow: hidden
  • Imagen: width: 100%, height: 100%, object-fit: cover
  • Border-radius: Esquinas redondeadas
  • Box-shadow: Sombra sutil
  • Transition: Para efectos hover suaves

✨ Tarea 4: Efectos Hover y Overlays

Implementa efectos interactivos avanzados:

  • Overlay: position: absolute, fondo semi-transparente
  • Información: Título, descripción, categoría
  • Estados:
    • Normal: overlay oculto
    • Hover: overlay visible con animación
    • Focus: accesible por teclado
  • Transformaciones: scale, translate para efectos dinámicos

💡 Ejemplo de Overlay Animado:

.gallery-item {
  position: relative;
  overflow: hidden;
  border-radius: var(--border-radius);
  transition: var(--transition);
}

.gallery-item:hover {
  transform: translateY(-5px);
  box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2);
}

.gallery-overlay {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  background: var(--overlay-bg);
  color: white;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  opacity: 0;
  transition: var(--transition);
}

.gallery-item:hover .gallery-overlay {
  opacity: 1;
}

🎭 Tarea 5: Animaciones CSS Avanzadas

Añade animaciones sofisticadas:

  • Entrada escalonada: animation-delay diferente para cada item
  • Efectos de imagen: scale, rotate, filter en hover
  • Texto animado: slide-in, fade-in para información
  • Botones: Efectos hover con transformaciones
💡 Pista: Usa @keyframes para animaciones personalizadas y :nth-child() para delays escalonados.

🏷️ Tarea 6: Sistema de Filtros

Implementa filtros de categorías (solo CSS):

  • Botones de filtro: All, Nature, Architecture, People
  • Data attributes: data-category en cada item
  • CSS targeting: [data-category="nature"] para mostrar/ocultar
  • Transiciones: Suaves al filtrar
🎨 Propiedades CSS Grid Clave
Propiedad Aplicar a Valores Uso en Galería
display: grid Container grid Contenedor principal
grid-template-columns Container repeat(auto-fit, minmax()) Columnas responsive
grid-auto-rows Container 200px, auto Altura de filas
grid-column Items span 2, 1 / 3 Elementos anchos
grid-row Items span 2, 1 / 4 Elementos altos
gap Container 1rem, 20px Espaciado entre items

📱 Tarea 7: Responsive Avanzado

Optimiza para diferentes dispositivos:

  • Desktop: 4-5 columnas, elementos destacados grandes
  • Tablet: 2-3 columnas, elementos destacados medianos
  • Mobile: 1-2 columnas, sin elementos destacados
  • Auto-fit: Usa minmax() para adaptación automática
🖼️ Categorías Sugeridas
  • Nature: Paisajes, animales, plantas
  • Architecture: Edificios, estructuras, diseño
  • People: Retratos, grupos, actividades
  • Abstract: Arte, patrones, conceptual
🚀 Desafío Extra: Añade un lightbox modal, implementa lazy loading con CSS, crea un modo masonry, o añade efectos de parallax.
